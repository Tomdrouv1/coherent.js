## Community Support

### GitHub Discussions

For questions and general discussion about Coherent.js, [GitHub Discussions](https://github.com/Tomdrouv1/coherent.js/discussions) is the best place to ask. You can:

- Ask questions about how to use Coherent.js
- Share ideas and get feedback
- Connect with other Coherent.js developers
- Show off your projects

### GitHub Issues

For reporting bugs and requesting features, please use [GitHub Issues](https://github.com/Tomdrouv1/coherent.js/issues). Before creating an issue, please:

1. Check if there's already an open issue for your problem
2. Make sure you're using the latest version of Coherent.js
3. Follow the issue templates to provide all necessary information

## Getting Help

When asking for help, please provide:

1. A clear description of your problem
2. The version of Coherent.js you're using
3. The version of Node.js you're using
4. A minimal code example that reproduces your issue
5. Any error messages you're seeing
