# @coherent.js/language-service

Language service plugin for Coherent.js.

## Features

- Enhanced autocomplete for Coherent.js object syntax (Work in Progress)
- Validation for component structure

## Setup

Add to your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "plugins": [
      { "name": "@coherent.js/language-service" }
    ]
  }
}
```
